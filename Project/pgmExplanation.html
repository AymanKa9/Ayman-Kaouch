Temperature Conversion Program

This program is designed to convert a temperature value from Fahrenheit to Celsius. It is a fundamental tool for users who need to understand weather reports or scientific data in metric units.

How the Program Works

The program takes a numerical input representing degrees Fahrenheit (f). It then applies the following mathematical formula to determine the Celsius (c) equivalent:

c = (f − 32) × 5/9

Key Variables

  • f: The input temperature in Fahrenheit.
  • c: The calculated temperature in Celsius.

Using this logic, if the input is 212°F, the output will be 100°C.
